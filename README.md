# Virtual Minds - Psychiatric Practice Management Platform

This is an premium, high-conversion "Storybrand" website platform specifically engineered for California Psychiatric Practices. It is designed to bridge the gap between clinical excellence and administrative efficiency.

## 🚀 Key Features
- **Storybrand Architecture:** Copy and layout optimized for maximum patient trust and conversion.
- **Service Silos:** Clear categorization of Clinical Ops, Virtual Support, and Revenue Protection.
- **Interactive Tools:** Built-in "Practice Leaks Calculator" to demonstrate ROI to prospects.
- **SEO Hardened:** Optimized for high-intent keywords like "California Psychiatric Virtual Assistant" and "HIPAA Compliant Practice Management."
- **HIPAA-Ready UI:** Professional, clinical aesthetics using the "Clinical Precision" design system.

## 🛠 Technical Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Backend/DB:** Convex
- **CMS:** Sanity.io
- **Analytics:** Plausible & Google Analytics (GDPR/HIPAA considerations)
- **Messaging:** Resend

## 📦 Getting Started

1. **Clone the repository.**
2. **Setup Environment:**
   ```bash
   cp .env.example .env.local
   ```
   *Fill in your own API keys for Sanity, Convex, Resend, and Plausible.*
3. **Install Dependencies:**
   ```bash
   npm install
   ```
4. **Run Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3001](http://localhost:3001) to view the result.
